Files are saved in [Home]/uploads/.
  • uploaded zip files can be unzipped on the device with
    (unzip "uploads/all.zip" "examples/") ; android
    (unzip "uploads/all.zip" "doc/examples/") ; iOS
     
  • files on the device can be zipped before downloading with
    (zip "all.zip" "examples/") ; android
    (zip "all.zip" "doc/examples/") ; iOS
     
    and downloaded like so:
    http://192.168.1.x:1701/all.zip
